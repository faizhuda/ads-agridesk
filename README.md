# Agridesk

Agridesk is an academic letter workflow system with a FastAPI backend and a frontend app (to be added in `frontend/`).

## Repository Structure

```text
ads-agridesk/
	backend/    # FastAPI API, database migrations, tests
	frontend/   # Frontend application (currently empty)
```

## Documentation

- Backend setup and API reference: `backend/README.md`

## Quick Start

1. Follow backend setup instructions in `backend/README.md`.
2. Run the backend API and verify `http://127.0.0.1:8000/docs` is available.
3. Build the frontend in `frontend/` and connect it to the backend base URL.
